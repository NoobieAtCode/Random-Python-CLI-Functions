### Functions

#### loadbar
Displays a loading bar onto the terminal.
##### Parameters
`barlength -> int`: `the length of the bar`<br>
`pause1 -> int`: `The length of the first pause`<br>
`pauselong -> int`: `The length of the long pause in the middle`<br>
`finalwait -> int`: `The length of the final wait`<br>
`color -> str`: `The color of the bar`<br>
`loading_text -> str`: `The text before the bar when it's loading`<br>
`load_finish -> str`: `The text that displays after it finishes`<br>
`end_cls -> bool`: `True: clears the terminal after finishing`<br>
`end_cls -> bool`: `True: prints new line after finishing`<br>
<hr>

#### tywr
Outputs a typewriting effect with the inputted text.
##### Parameters
`i -> str`: `The text to output`<br>
`s -> int`: `The speed of the effect`<br>
`enable_sys -> bool`: `True: uses sys_print and sys.stdout.flush instead of print`<br>
